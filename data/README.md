# Data

Only synthetic, public, or explicitly authorized traces belong here.

`sample_traces.jsonl` is a deterministic teaching fixture. It is intentionally small and must not be presented as a representative benchmark.

Every future dataset version should document:

- provenance and license
- collection or generation process
- label schema and annotation instructions
- class distribution
- duplicate and leakage checks
- train, validation, test, and out-of-distribution split logic
- privacy and redaction controls
